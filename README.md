\# SmartSplit (Backend)



Spring Boot backend for a Splitwise-like expense sharing app.



\## Tech

\- Java 21 / 24

\- Spring Boot 4.x

\- Spring Security (JWT)

\- Spring Data JPA (Hibernate)

\- MySQL



\## Features

\- Auth (register/login)

\- Groups \& members

\- Expenses \& splits

\- Balance calculation

\- Settlement suggestions (minimize payments)

\- Mark settlement as PAID

\- Notifications (mark as read)

\- Activity logs
